Browser Speech Worker

Browser Recognition Window

This worker captures microphone audio with the browser Speech Recognition API and forwards partial/final text to the local subtitle app over WebSocket.

Keep This Window Visible

Browser Speech can become unstable if this window is minimized, fully hidden, or left in a throttled background state for too long. For the most reliable recognition, keep this window open and visible while it is listening.
