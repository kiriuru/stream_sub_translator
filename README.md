# SST Desktop 2.7.3

Локальное desktop-приложение для распознавания и перевода речи в субтитры в реальном времени.

## Что в релизе
- `Stream Subtitle Translator.exe`
- папка `app-runtime/`

Это минимальный рабочий комплект для запуска.

## Быстрый старт
1. Распакуйте архив в отдельную папку.
2. Убедитесь, что рядом находятся:
   - `Stream Subtitle Translator.exe`
   - `app-runtime/`
3. Запустите `Stream Subtitle Translator.exe`.
4. В стартовом окне выберите режим запуска.
5. После старта откроется основное окно SST.

## Режимы запуска
- `Quick Start`  
  Быстрый запуск через Browser Speech.
- `NVIDIA GPU (CUDA)`  
  Локальное AI-распознавание с приоритетом GPU.
- `CPU-only`  
  Локальное AI-распознавание без GPU.

## Основные возможности
- Распознавание речи с микрофона в реальном времени.
- Перевод в 0..N языков (по настройкам).
- Гибкий вывод субтитров:
  - только оригинал
  - только перевод
  - оригинал + один/несколько переводов
- OBS overlay через локальный URL.
- Экспорт сессий в `SRT` и `JSONL`.

## OBS Overlay
Базовый URL:
- `http://127.0.0.1:8765/overlay`

Поддерживаются query-параметры:
- `?profile=default`
- `?compact=1`

## Где хранятся данные
Рядом с `exe`:
- `user-data/`
  - `config.json`
  - `profiles/`
  - `exports/`
  - `models/`
  - `cache/`
- `logs/`
  - `desktop-launcher.log`
  - `overlay-events.log`
  - `browser-recognition.log`
  - `dashboard-live-events.log`

## Системные требования
- Windows 10/11 x64
- Рабочий микрофон
- Для GPU-режима: NVIDIA GPU + совместимый CUDA runtime
- Для режимов с внешним переводом: доступ в интернет и валидные API-ключи провайдера

## Поведение первого запуска
- Первый запуск может занимать больше времени.
- Возможна дополнительная инициализация локального runtime и моделей.
- В процессе используйте логи из папки `logs/`.

## Обновление релиза
Чтобы обновиться на новую версию:
1. Остановите SST.
2. Замените:
   - `Stream Subtitle Translator.exe`
   - папку `app-runtime/`
3. `user-data/` и `logs/` можно сохранить (это ваши локальные данные и история).

## Быстрый troubleshooting
- Ошибка запуска: проверьте, что `app-runtime/` находится рядом с `exe`.
- Нет доступа к интерфейсу: убедитесь, что порт `8765` не занят другим процессом.
- Нет текста в Browser Speech: проверьте, что установлен Chrome/Edge и разрешен доступ к микрофону.
- Медленный старт AI-режима: дождитесь первичной инициализации модели.
