      Quick Tuning
      
              
      Recognition Feel
      
              
      Save config or profile after tuning.
      
            
      
          
      
          
      
            
            
      Presets set backend realtime fields (streaming, word growth). Sliders switch to Custom when you fine-tune manually.
      
            
      
              After changing presets or sliders: save config or profile, then restart the runtime (Stop, then Start) so local Parakeet reloads realtime settings.
            
      
          
      
          
      
            
            
      Higher values show in-progress text sooner. Lower values wait for a bit more speech.
      
          
      
          
      
            
            
      Higher values finalize phrases faster after a pause. Lower values wait longer before closing the subtitle block.
      
          
      
          
      
            
            
      Higher values reduce tiny partial rewrites. Lower values feel more live but can flicker more.
      
          
      
          
      
            
          
      
          
      
            
            
      0% keeps the raw path. Higher values blend in more RNNoise output.
      
          
      
          
      Optional and off by default. It can reduce false recognition from constant background noise, but it may also change recognition behavior.

Does not change how ASR itself works.
