Web Speech Worker

Web Speech Window (Experimental)

This experimental worker opens one live microphone MediaStreamTrack and tries SpeechRecognition.start(audioTrack). If that is rejected, it can fall back to normal start().

Experimental mode is more sensitive to browser throttling. Keep this window open and visible while it is listening, or the audio-track path and quick rearms may become unstable.
